Example add bootstrap or css/scss file in app

Example add bootstrap

  1. Command in console app: npm install bootstrap
  2. in file style.scss add "@import '~bootstrap/dist/css/bootstrap.min.css';"

Example add global application *.css or *.scss file

  1. copy file *.scss or *.css in app example path: './src/assets/css'
  2. add file style.scss "@import './assets/css/*.css(*.scss)';"
